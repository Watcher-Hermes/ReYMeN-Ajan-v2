# Güvenlik Politikası / Security Policy

## Desteklenen Sürümler / Supported Versions

| Version | Supported |
|---------|-----------|
| 2.1.x   | ✅ |
| 2.0.x   | ⚠️ Sadece kritik güvenlik |
| < 2.0   | ❌ |

## Güvenlik Açığı Bildirimi / Reporting a Vulnerability

**Kritik güvenlik açıkları için lütfen HERKESE AÇIK ISSUE AÇMAYIN.**

1. 📧 E-posta: **marko@reymen.dev**
2. 🔒 Konu satırına  ekleyin
3. ⏱ 48 saat içinde yanıt alırsınız
4. 🔧 Düzeltme yayınlanana kadar açık paylaşılmaz

### Bildirmeniz gerekenler:
- API key / token sızıntıları
- Remote code execution (RCE)
- Authentication bypass
- SQL injection
- Path traversal

### Bildirmeniz GEREKMEYENLER:
- Genel sorular / feature request
- Linting hataları
- Test coverage eksiklikleri
