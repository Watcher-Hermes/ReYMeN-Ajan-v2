Tarih: 2026-06-14

## En Kapsamlı Çözümler

### 1. ReYMeN-studio (7.890★)
- https://github.com/EKKOLearnAI/ReYMeN-studio
- Web dashboard + Desktop app (Windows/macOS/Linux)
- npm ile kurulum: `npm install -g ReYMeN-web-ui && ReYMeN-web-ui start`
- Vue3 + TypeScript
- Chat, session, profile, cron, file browser, terminal, analytics
- Platform channels: Telegram, Discord, Slack, WhatsApp, Matrix, Feishu, WeChat, WeCom

### 2. ReYMeN-control-interface (762★)
- https://github.com/xaspx/ReYMeN-control-interface
- Self-hosted web dashboard (localhost:10274)
- Vanilla JS + Vite + Node.js + WebSocket
- Chat, agents, MCP control, skills browser, token analytics, backup
- `git clone + npm install + node server.js`

### 3. ReYMeN-Studio (195★)
- https://github.com/JPeetz/ReYMeN-Studio
- React + TypeScript + PWA
- Multi-agent orchestration, memory, skills, approvals
- Self-hosted

### 4. hermesd (92★)
- https://github.com/mudrii/hermesd
- Python TUI (terminal içi dashboard)
- Hafif, bağımlılık yok

### 5. ReYMeN-webui (106★)
- https://github.com/sanchomuzax/ReYMeN-webui
- Process monitoring + configuration dashboard

## Seçim Kriterleri
- Windows'ta çalışmalı → ReYMeN-studio (desktop app) veya ReYMeN-control-interface (web)
- Kurulum kolaylığı → ReYMeN-studio (tek npm komutu)
- Kapsam → ReYMeN-studio (en geniş özellik seti)
